# 🥗 Smart Nutrition - تطبيق التغذية الذكية

تطبيق Flutter متقدم لحساب التغذية والسعرات الحرارية مع نصائح تغذية ورياضة وإشعارات ذكية.

## ✨ الميزات الرئيسية

### 📊 حساب التغذية
- حساب BMI والاحتياجات الغذائية اليومية
- تتبع السعرات الحرارية والبروتين والكربوهيدرات والدهون
- تقارير مفصلة بصيغة PDF

### 📸 ماسح الطعام
- ماسح ضوئي للطعام باستخدام الكاميرا
- التعرف على الطعام باستخدام Google Vision API
- اقتراحات ذكية للطعام

### 🔔 نظام الإشعارات
- إشعارات فورية عند إضافة وجبة
- عرض السعرات الحرارية والبروتين
- إشعارات مجدولة يومية

### 💡 نصائح التغذية والرياضة
- 30 نصيحة متنوعة
- نصائح عشوائية يومية
- نصائح مخصصة حسب نوع الوجبة

### 🎨 واجهة مستخدم جميلة
- دعم كامل للعربية
- وضع مظلم وفاتح
- تصميم حديث وسهل الاستخدام
