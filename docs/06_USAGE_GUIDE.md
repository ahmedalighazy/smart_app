## 🚀 البدء السريع

### 1. تثبيت المتطلبات

```bash
# تثبيت Flutter
flutter doctor

# تثبيت المكتبات
flutter pub get

# تشغيل build_runner لـ Hive
flutter pub run build_runner build --delete-conflicting-outputs
```

### 2. إعداد API Keys

**Google Gemini API**:
```dart
// في lib/data/services/food_vision_service.dart
final String apiKey = 'YOUR_GEMINI_API_KEY';
```

**الحصول على المفتاح**:
1. اذهب إلى: https://makersuite.google.com/app/apikey
2. سجل دخول بحساب Google
3. اضغط "Create API Key"
4. انسخ المفتاح وضعه في الكود

### 3. تشغيل التطبيق

```bash
# Android
flutter run

# iOS
flutter run -d ios

# Web
flutter run -d chrome
```

## 👤 إدارة الحساب

### تسجيل حساب جديد

1. افتح التطبيق
2. اضغط على "إنشاء حساب"
3. أدخل البيانات:
   - الاسم الكامل
   - البريد الإلكتروني
   - كلمة المرور (6 أحرف على الأقل)
   - تأكيد كلمة المرور
4. اضغط "إنشاء حساب"

**مثال**:
```
الاسم: أحمد محمد
البريد: ahmed@example.com
كلمة المرور: 123456
```

### تسجيل الدخول

1. افتح التطبيق
2. أدخل البريد الإلكتروني وكلمة المرور
3. اضغط "تسجيل الدخول"

**ملاحظة**: التطبيق يحفظ الجلسة تلقائياً، لن تحتاج لتسجيل الدخول في كل مرة.

### تسجيل الخروج

1. اذهب إلى "الإعدادات"
2. اضغط على "تسجيل الخروج"
3. أكد العملية

## 📊 حساب الاحتياجات الغذائية

### الخطوات:

1. **من الصفحة الرئيسية**:
   - اضغط على "حاسبة التغذية"

2. **أدخل بياناتك**:
   - العمر (سنة)
   - الطول (سم)
   - الوزن (كجم)
   - الجنس (ذكر/أنثى)
   - مستوى النشاط:
     * قليل جداً: لا رياضة
     * خفيف: 1-3 أيام/أسبوع
     * معتدل: 3-5 أيام/أسبوع
     * نشط: 6-7 أيام/أسبوع
     * نشط جداً: رياضي محترف
   - الحالة الفسيولوجية (اختياري):
     * عادي
     * حامل
     * مرضعة
     * رياضي محترف

3. **اضغط "احسب الآن"**

4. **عرض النتائج**:
   - BMI (مؤشر كتلة الجسم)
   - BMR (معدل الأيض الأساسي)
   - TDEE (السعرات اليومية المطلوبة)
   - توزيع العناصر الغذائية:
     * الكربوهيدرات (جرام)
     * البروتين (جرام)
     * الدهون (جرام)

### مثال عملي:

```
المدخلات:
- العمر: 25 سنة
- الطول: 175 سم
- الوزن: 70 كجم
- الجنس: ذكر
- النشاط: معتدل

النتائج:
- BMI: 22.9 (وزن طبيعي)
- BMR: 1,680 سعرة
- TDEE: 2,604 سعرة
- الكربوهيدرات: 292g
- البروتين: 163g
- الدهون: 87g
```

## 📸 مسح الطعام بالكاميرا

### الخطوات:

1. **اذهب إلى تبويب "الماسح"**

2. **اختر طريقة الإدخال**:
   - "التقط صورة": فتح الكاميرا
   - "اختر من المعرض": اختيار صورة موجودة

3. **التقط/اختر صورة الطعام**

4. **انتظر التحليل** (5-10 ثواني)

5. **عرض النتائج**:
   - اسم الطعام
   - المكونات
   - حجم الحصة
   - السعرات الحرارية
   - البروتين
   - الكربوهيدرات
   - الدهون
   - الألياف
   - التقييم الصحي
   - نصائح
   - تحليل تفصيلي

### نصائح للحصول على أفضل نتائج:

✅ **افعل**:
- التقط صورة واضحة
- إضاءة جيدة
- الطعام في المنتصف
- زاوية مباشرة

❌ **لا تفعل**:
- صور مظلمة
- صور بعيدة جداً
- صور غير واضحة
- عدة أطعمة مختلفة في صورة واحدة

### مثال:

```
الصورة: دجاج مشوي مع أرز

النتائج:
- الطعام: دجاج مشوي مع أرز بني
- المكونات: صدر دجاج، أرز بني، زيت زيتون، بهارات
- الحجم: متوسط
- السعرات: 450 سعرة
- البروتين: 35g
- الكربوهيدرات: 45g
- الدهون: 12g
- التقييم: جيد ✅
- النصيحة: وجبة متوازنة، أضف خضروات للمزيد من الألياف
```

## 🍽️ إدارة الوجبات

### إضافة وجبة يدوياً:

1. **من الصفحة الرئيسية**:
   - اضغط على فئة الطعام (لحوم، حبوب، ألبان، إلخ)

2. **اختر الطعام**:
   - تصفح قائمة الأطعمة
   - أو استخدم البحث

3. **اختر نوع الوجبة**:
   - فطور
   - غداء
   - عشاء
   - وجبة خفيفة

4. **اضغط "إضافة"**

### عرض الوجبات:

1. **اذهب إلى تبويب "وجباتي"**

2. **عرض الوجبات حسب**:
   - اليوم
   - نوع الوجبة
   - الفئة

3. **عرض الإحصائيات**:
   - إجمالي السعرات
   - إجمالي البروتين
   - إجمالي الكربوهيدرات
   - إجمالي الدهون

### حذف وجبة:

1. اضغط على الوجبة
2. اضغط على أيقونة الحذف 🗑️
3. أكد الحذف

### مثال يوم كامل:

```
الفطور (8:00 صباحاً):
- بيض مسلوق (2 حبة): 140 سعرة، 12g بروتين
- خبز أسمر (شريحتين): 160 سعرة، 6g بروتين
- حليب قليل الدسم: 100 سعرة، 8g بروتين
الإجمالي: 400 سعرة، 26g بروتين

الغداء (1:00 ظهراً):
- دجاج مشوي (150g): 250 سعرة، 30g بروتين
- أرز بني (كوب): 220 سعرة، 5g بروتين
- سلطة خضراء: 50 سعرة، 2g بروتين
الإجمالي: 520 سعرة، 37g بروتين

العشاء (7:00 مساءً):
- سمك مشوي (120g): 200 سعرة، 25g بروتين
- خضار مطبوخة: 80 سعرة، 3g بروتين
الإجمالي: 280 سعرة، 28g بروتين

وجبة خفيفة (4:00 عصراً):
- تفاحة: 95 سعرة، 0.5g بروتين
- لوز (30g): 170 سعرة، 6g بروتين
الإجمالي: 265 سعرة، 6.5g بروتين

إجمالي اليوم:
- السعرات: 1,465 سعرة
- البروتين: 97.5g
- الكربوهيدرات: 145g
- الدهون: 45g
```

## 🤖 الحصول على توصيات AI

### الخطوات:

1. **تأكد من إدخال بياناتك**:
   - يجب حساب احتياجاتك أولاً

2. **من الصفحة الرئيسية**:
   - اضغط على "توصيات AI"

3. **انتظر التحليل** (10-15 ثانية)

4. **عرض التوصيات**:
   - تقييم الحالة الصحية
   - نصائح غذائية محددة
   - أمثلة على وجبات
   - أطعمة مُنصح بها
   - أطعمة يجب تجنبها

5. **تحديث التوصيات**:
   - اضغط على أيقونة التحديث 🔄

### مثال توصيات:

```
📊 تقييم الحالة الصحية:
وزنك طبيعي (BMI: 22.9)، واحتياجك اليومي 2,604 سعرة.
مستوى نشاطك معتدل، وهذا جيد للحفاظ على الصحة.

💡 نصائح غذائية:
1. تناول 5-6 وجبات صغيرة بدلاً من 3 كبيرة
2. اشرب 2.5 لتر ماء يومياً
3. أضف البروتين في كل وجبة
4. تناول الفواكه والخضروات بألوان مختلفة
5. قلل من السكريات المضافة

🍽️ مثال على وجبات يومية:

الفطور:
- شوفان مع حليب وموز
- بيض مسلوق
- عصير برتقال طبيعي

الغداء:
- دجاج مشوي
- أرز بني
- سلطة خضراء
- زبادي

العشاء:
- سمك مشوي
- خضار مطبوخة
- خبز أسمر

✅ أطعمة يُنصح بها:
- البروتين: دجاج، سمك، بيض، لحم أحمر قليل الدهن
- الكربوهيدرات: أرز بني، شوفان، خبز أسمر، بطاطا حلوة
- الدهون الصحية: مكسرات، أفوكادو، زيت زيتون
- الخضروات: بروكلي، سبانخ، جزر، طماطم
- الفواكه: تفاح، موز، برتقال، توت

❌ أطعمة يُنصح بتجنبها:
- الوجبات السريعة
- المشروبات الغازية
- الحلويات المصنعة
- الأطعمة المقلية
- الأطعمة عالية الصوديوم
```

## 💡 النصائح اليومية

### الوصول للنصائح:

1. **اذهب إلى تبويب "نصائح"**

2. **عرض النصائح**:
   - نصائح غذائية
   - نصائح رياضية
   - نصائح عامة

3. **الحصول على نصيحة جديدة**:
   - اضغط على "نصيحة جديدة"

### الإشعارات التلقائية:

- التطبيق يرسل نصيحة كل دقيقة تلقائياً
- يمكنك إيقاف الإشعارات من الإعدادات

### أمثلة على النصائح:

**نصائح غذائية**:
- اشرب 8 أكواب ماء يومياً
- تناول الخضروات الورقية الخضراء
- قلل من السكريات والدهون المشبعة
- تناول البروتين في كل وجبة
- لا تتخطى وجبة الإفطار

**نصائح رياضية**:
- مارس الرياضة 30 دقيقة يومياً
- ابدأ بتمارين الإحماء
- اشرب ماء قبل وأثناء وبعد التمرين
- مارس تمارين القوة 3 مرات أسبوعياً
- لا تنسى تمارين الاستطالة

## ❓ حل المشاكل الشائعة

### 1. لا يعمل مسح الطعام

**الحل**:
- تأكد من إضافة Gemini API Key
- تحقق من اتصال الإنترنت
- تأكد من صلاحيات الكاميرا

### 2. لا تظهر الإشعارات

**الحل**:
- تحقق من صلاحيات الإشعارات
- تأكد من تفعيل الإشعارات في الإعدادات
- أعد تشغيل التطبيق

### 3. لا تُحفظ الوجبات

**الحل**:
- تأكد من تهيئة Hive بشكل صحيح
- أعد تشغيل التطبيق
- امسح cache التطبيق

### 4. خطأ في تسجيل الدخول

**الحل**:
- تحقق من البريد الإلكتروني وكلمة المرور
- تأكد من اتصال الإنترنت
- تحقق من عمل الـ API
